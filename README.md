# norik_muransky_microbiome
16S rRNA V3–V4 skin microbiome of 12 healthy Norik Muránský draught mares at 8 body sites.  Ground-contact vs elevated topology sites classified by random forest (OOB 97.9%, AUC 0.999).    ▎ Includes 4-family probe panel, LMM sensitivity analyses, PICRUSt2 pathway profiling, and eczema hypothesis figure.
